# Typescript template

## Instructions

Use [degit](https://github.com/Rich-Harris/degit) to start a new project using this template

```bash
npx degit nkhil/ts-template <new-project-name>
```

## Setup

Install dependencies

```sh
npm install
```

Start compiling Typescript to javascript

```sh
npm run build:watch
```

Run tests

```sh
npm t
```
